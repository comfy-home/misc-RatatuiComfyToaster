# Contributing to RatatuiComfyToaster

Thank you for your interest in contributing to RatatuiComfyToaster!  
This project welcomes improvements, bug fixes, and enhancements, provided they align with the
project’s goals and licensing model.

Please read this document carefully before submitting Issues or Pull Requests.

---

## 1. Code of Conduct
By participating in this project, you agree to:
- Be respectful and constructive
- Communicate clearly and professionally
- Avoid hostile, aggressive, or disruptive behaviour

We aim to maintain a collaborative and welcoming environment.

---

## 2. Reporting Issues
If you encounter bugs, unexpected behaviour, or usability problems, please open an Issue on GitHub.

Your Issue **must** include:
- A clear description of the problem
- Steps to reproduce
- Expected vs. actual behaviour
- Environment details (OS, Rust version, etc.)
- Logs or screenshots if relevant

Submitting an Issue does not obligate ComfyHome™ to provide support, but it ensures problems are
tracked in the correct place.

---

## 3. Submitting Pull Requests
Pull Requests (PRs) are welcome and appreciated.  
By submitting a PR, you agree to the following:

### 3.1 Contribution License Agreement
All contributions are:
- Your original work
- Licensed to ComfyHome™ under the project’s **Source-Available / Protected-Source — Distribution Allowed (SA-PS:DA)** license
- Assigning ComfyHome™ the right to:
  - Use your contribution
  - Modify it
  - Distribute it
  - Re-license it under commercial terms if needed

This ensures the project remains legally consistent and maintainable.

### 3.2 PR Requirements
Before submitting a PR:
- Ensure your code builds without errors
- Run formatting (`cargo fmt`) and linting (`cargo clippy`) if applicable
- Add or update tests when appropriate
- Keep changes focused and atomic
- Update documentation if your change affects behaviour

### 3.3 PR Review Process
All PRs are reviewed for:
- Code quality
- Consistency with project goals
- Licensing compliance
- Security considerations
- Maintainability

ComfyHome™ reserves the right to request changes or decline contributions.
